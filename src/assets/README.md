# Assets Directory

This directory contains the non-code resources used by the slot machine application.

## Purpose

The assets folder stores media and supporting files that are referenced by the app, including:

- images
- icons
- sound effects
- visual decorations
- other static resources

## Suggested Structure //Change as you go as this is placeholder structure


- `images/` — background images, illustrations, screenshots, decorative graphics
- `icons/` — symbols, UI icons, reel icons, button graphics
- `sounds/` — spin sounds, win/loss sounds, button clicks, ambient effects

## Expectations

Assets included here should:

- support the overall visual and thematic direction of the project
- be organized clearly by type
- use meaningful file names
- avoid unnecessary duplicates or unused files

## Notes

If assets are AI-generated, externally sourced, or modified manually, that information should be documented where appropriate in the project notes or logs.
