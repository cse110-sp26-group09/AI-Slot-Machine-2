# Scripts Directory

This directory contains the JavaScript source files for the slot machine application.

## Purpose

The scripts folder organizes the game logic into smaller, focused modules rather than placing everything into one file.

## Suggested Structure //Change as you go as this is placeholder structure

- `main.js` — application startup and initialization
- `game.js` — overall game flow and state management
- `reels.js` — reel spinning logic and symbol selection
- `payouts.js` — payout rules and reward calculations
- `ui.js` — DOM updates, controls, and display logic
- `audio.js` — sound effects and audio management

## Expectations

Code in this directory should aim to follow software engineering best practices:

- meaningful variable and function names
- small and focused functions
- minimal duplication
- clear separation of concerns
- readable and maintainable structure
- error handling where needed

## Quality Standards

Scripts should be:

- linted regularly
- documented where appropriate
- tested at the unit level when possible
- kept consistent in style and structure
