# Docs Directory

This directory holds supporting documentation for the AI Slot Machine project development process.

## Purpose

The docs folder captures technical and process-related information that supports the project but doesn't fit in source code or planning artifacts.

## Contents

- `prompting-notes.md` — Records prompting patterns, observations, and lessons learned from working with AI coding assistants

## Role in the Project

This documentation helps track:

- Effective prompting strategies and techniques
- Insights from AI-assisted development
- Process improvements and learnings

## Expectations

Documentation here should be:

- Clear and accessible
- Updated as the project progresses
- Specific about decisions and outcomes
- Valuable for team members, reviewers, and future reference

## Notes

This folder supports the planning in `plan/` and conclusions in `final-report/`, providing context for the engineering process and AI usage.
