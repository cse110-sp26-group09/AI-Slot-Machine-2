User Stories

Researcher: Waleed  
Date: April 20, 2026

1. As a new player, I want to easily access an interactive paytable and rules breakdown so that I understand exactly how winning lines and bonus features work. *(clarity of game mechanics)*

2. As an entertainment-focused player, I want distinct, escalating audio-visual celebrations for larger wins so that my successes feel impactful and exciting. *(satisfying feedback)*

3. As a mobile user on a commute, I want the spin button and bet modifiers clustered tightly together at the bottom of the screen so that I can easily control the game with one hand. *(ease of use)*

4. As a player with fine motor challenges, I want an option to enable auto-spin with clear, customizable stop conditions so that I don't have to repeatedly tap the screen to play. *(accessibility of controls)*

5. As a regular visitor, I want to fill a persistent visual progression meter during base gameplay that unlocks a special feature so that my time investment always feels rewarding. *(replay value and engagement)*
