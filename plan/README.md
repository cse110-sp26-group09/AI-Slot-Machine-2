# Plan Directory

This directory contains the planning and research artifacts for the AI Slot Machine SWE Warm-Up.

## Purpose

The goal of this directory is to show our team’s thinking before and during development. It documents:

- Raw research on slot machine products, features, themes, and terminology
- User-centered design artifacts such as personas and user stories
- Our initial AI workflow strategy
- Our ongoing log of how AI was actually used during implementation

## Files

- `research-overview.md` — summary of research and team contributions
- `ai-plan.md` — initial strategy for using the coding assistant
- `ai-use-log.md` — running log of prompts, results, decisions, and lessons
- `personas/` — user personas used to guide design decisions
- `user-stories/` — user stories tied to product behavior and acceptance
- `raw-research/` — collected research notes, feature ideas, market examples, and terminology

## Why this matters

This assignment emphasizes research, planning, and documentation before and during development. These artifacts show how our team grounded the product in both software engineering process and user-centered thinking.
