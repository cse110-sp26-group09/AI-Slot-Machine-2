# Plan Directory

This directory contains planning, research, and design artifacts for the AI Slot Machine project.

## Purpose

The plan directory captures the team’s process, design decisions, and AI usage strategy before and during implementation.

It includes:

- research findings and product context
- personas that represent target users
- user stories that describe desired behaviors
- an AI strategy document and usage log

## Contents

- `research-overview.md` — research summary and team conclusions
- `ai-plan.md` — planned approach for using AI in development
- `ai-use-log.md` — chronological record of prompts, responses, edits, and decisions
- `personas/` — user persona artifacts
  - `Daniel-personas.md`
  - `josh-personas.md`
  - `waleeds_persona.md`
  - `woosik-personas.md`
  - `README.md`
- `user-stories/` — product requirements expressed as user stories
  - `user-stories.md`
  - `README.md`
- `raw-research/` — supporting research notes and raw inputs
  - `daniel-research.md`
  - `josh-research.md`
  - `waleeds_research.md`
  - `README.md`

## How to use this directory

- Start with `research-overview.md` to understand the project context.
- Review `personas/` and `user-stories/` for user-centered requirements.
- Use `ai-plan.md` to see how the team intended to use AI responsibly.
- Follow `ai-use-log.md` to track the actual prompts, changes, and learning.

## Why it matters

These artifacts show how the team organized planning work, grounded decisions in research, and documented AI-assisted development for the project.
